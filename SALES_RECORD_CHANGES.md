# Phase 3.1.5 — Sales Record

## Changes
1. The visible module name is now **Sales Record**.
2. Imported sales can be deleted by users with Financials write permission.
3. A confirmation explains the impact before deletion.
4. If the sale created a linked Financials income entry, that linked entry is deleted as well.
5. Customer records and import batch history are retained.
6. The Audit Log records invoice, branch, amount and whether a linked financial entry was removed.
