# Mobile, iPhone, iPad, Tablet aur Windows Installation

Public HTTPS demo link generate hone ke baad ye steps use karein.

## Android Phone / Android Tablet

1. Link Google Chrome mein open karein.
2. Chrome menu `⋮` press karein.
3. **Install app** ya **Add to Home screen** select karein.
4. **Install** confirm karein.
5. Dar al Sultan golden logo home screen par aa jayega.

## iPhone / iPad

1. Link Safari mein open karein.
2. Share icon press karein.
3. **Add to Home Screen** select karein.
4. Available ho to **Open as Web App** enable karein.
5. **Add** press karein.

## Windows Laptop/Desktop

1. Link Chrome ya Edge mein open karein.
2. Address bar ka install icon press karein, ya browser menu se **Install Dar al Sultan** select karein.
3. Install confirm karein.
